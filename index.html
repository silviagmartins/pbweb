Programação básica para WEB

Olá, meu nome é Mauro e seja bem-vindo à disciplina Programação básica para WEB. Essa disciplina apresentará, por meio da aplicação de padrões do W3C:

  • O que é e como usar a linguagem de marcação HyperText Markup Language (HTML);
  • Construção de estilos para páginas Web com Cascade Style Sheet (CSS);
  • Introdução de comportamentos em páginas Web por meio do uso de programação em Javascript.
